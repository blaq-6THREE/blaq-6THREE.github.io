GIT

What is GIT?

Git is an open source VCS(Version Control System). What it basically means is it keeps track of the various changes done to a file and folder once the changes are STAGED and PUSHED. Do not worry about the bolded text. We will get to understanding them later.

Why GIT?

I will not get into the history of VCS but rather state a couple of reasons why you would want to use GIT.

  • Allows for perfect collaboration
  • There is good documentation for it
  • Easy to use
  • Easy to revert mistakes.
  • Branches, which we will get to allows for developing in parallel to the main repository.

Getting Started

There are various ways to using GIT but we will be using the Command-Line Tools. I will expect you to know how to open Terminal in Mac or Command Prompt or Powershell in Windows. If you don't know what these are then stop here, go do research and come back.

Installing GIT on Mac

To install git on Mac OS run:

$ git --version

What this above line does is check if Git is installed. If it is, the current version of Git is returned. If not, it will prompt you to install it.
